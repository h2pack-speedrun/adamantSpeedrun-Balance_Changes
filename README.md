# Balance Changes

Balance change options for the Speedrun modpack.

Part of the [Speedrun modpack](https://thunderstore.io/c/hades-ii/p/adamant/Speedrun_Modpack/).

## What It Does

Balance Changes collects optional fixes and rule changes that alter weapon, boon, encounter, or damage behavior for speedrun consistency.

Current options:

- Braid Fix
- Cardio Torch Fix
- ET Fix
- Omega Cast Fix
- Poseidon Waves Fix
- Remove Second Channeling
- Shimmering Moonshot Fix
- Extra Dose Fix
- Aspect of Selene Fix
- Axe and Blade Omega Channel Fix
- Tidal Ring Fix
- GGG Echo Fix
- Suffering Fix

## Gameplay Impact

These options can affect build strength, boon behavior, weapon behavior, or encounter damage rules. Each option is individually configurable from the shared Speedrun settings window.

## Installation

Install via r2modman.

This module is usually installed as part of the full [Speedrun modpack](https://thunderstore.io/c/hades-ii/p/adamant/Speedrun_Modpack/), where it appears in the shared Speedrun UI with the other speedrun-focused modules.
